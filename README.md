✅ **PROJECT 3 COMPLETE**

## Overview

Industrial communication using Siemens TIA Portal + PLCSIM Advanced 4.0. Two virtual PLCs communicate via GET/PUT (S7) and MODBUS TCP.

**Programming Language:** Ladder Logic

---

## Protocols Covered

| Protocol | Type | Use |
|----------|------|-----|
| GET/PUT | Siemens S7 | PLC to PLC data exchange |
| MODBUS TCP | Open Standard | Cross-vendor communication |

---

## System Setup

- **Software:** TIA Portal V16 + PLCSIM Advanced 4.0
- **Instances:** 2 Virtual PLCs
- **Language:** Ladder Logic

---

## Ladder Logic Implementation

### GET/PUT Communication

| Network | Description |
|---------|-------------|
| Network 1 | Enable GET/PUT communication between PLCs |
| Network 2 | GET instruction - Read data from remote PLC |
| Network 3 | PUT instruction - Write data to remote PLC |
| Network 4 | Error handling and status monitoring |

### MODBUS TCP

| Network | Description |
|---------|-------------|
| Network 1 | MODBUS Client configuration |
| Network 2 | MODBUS Server configuration |
| Network 3 | Read holding registers |
| Network 4 | Write holding registers |
